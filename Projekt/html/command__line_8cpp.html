◆ isNumber()

bool isNumber ( const std::string &  s)

Pomocnicza funkcja sprawdzająca, czy podany ciąg znaków jest liczbą całkowitą.

  • Używana do walidacji argumentów określających systemy liczbowe.
  • Parametry
    sCiąg znaków do sprawdzenia.
    Zwraca
    true Jeśli ciąg składa się wyłącznie z cyfr.
    false Jeśli ciąg zawiera inne znaki.

◆ parseCommand()

std::optional< Arguments > parseCommand ( int  argc,
char *  argv[] 
)

Analizuje argumenty wiersza poleceń przekazane do programu.

  • Funkcja weryfikuje poprawność flag, sprawdza istnienie pliku wejściowego oraz waliduje format podanych systemów liczbowych. Jeśli uruchomiono program bez argumentów, wyświetla instrukcję obsługi.
  • Parametry
    argcLiczba argumentów (argument count).
    argvTablica argumentów (argument vector).
    Zwraca
    std::optional<Arguments> Wypełniona struktura z ustawieniami lub std::nullopt w przypadku błędu walidacji.
